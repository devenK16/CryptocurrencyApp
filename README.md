# Cryptocurrency App
  This is a simple CryptocurrencyApp which shows list of cryptocurrencies and their detials.

# Features
* MVVM + clean Architecture
* Jetpack Compose
* HILT
* Coroutines
* API Integration using Retrofit
* Error Handling
